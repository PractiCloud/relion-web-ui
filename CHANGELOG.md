# Changelog

## v1.0.0 - Initial public release

- First open-source release of the RELION Web UI.
- Three deployment variants: Local (single-machine), Headless (Slurm without OOD), and OOD (Slurm with Open OnDemand).
- Full RELION 5 single-particle analysis pipeline through a browser interface.
- Includes secondary React app for particle picking.
- Apache 2.0 license.
